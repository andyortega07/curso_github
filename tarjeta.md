# Objetivo profesional a 10 años

## Meta
En 10 años quiero trabajar como Desarrollador Mobile (Flutter o React Native) en una empresa internacional ubicada en Europa o Estados Unidos, en un rol de nivel mid o senior.

## Hitos (ejemplo)
- 1–2 años: Dominar Flutter o React Native; publicar 2 apps pequeñas; alcanzar nivel de inglés B2.
- 3–5 años: Obtener experiencia laboral remunerada (prácticas o puesto junior); tener 3–5 proyectos en el portafolio.
- 6–8 años: Rol mid/senior con responsabilidad técnica en proyectos móviles.
- 9–10 años: Conseguir posición en empresa internacional y, si aplica, reubicación.

## Habilidades a desarrollar
- Flutter o React Native (elige uno para especializarte)
- JavaScript / TypeScript / Dart
- Control de versiones (Git), pruebas unitarias y de UI, CI/CD
- APIs REST/GraphQL, bases de datos móviles
- Inglés técnico y conversacional (meta: C1)

## Acciones inmediatas (6–12 meses)
- Hacer un curso intensivo de Flutter o React Native y completar un proyecto guiado.
- Publicar al menos 2 proyectos en GitHub y, si es posible, en la tienda (App Store / Play Store).
- Mejorar LinkedIn y CV en inglés; preparar versión en español e inglés.
- Aplicar a prácticas remotas o locales y participar en comunidades (foros, meetups).
